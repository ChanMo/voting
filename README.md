# Voting Dapp Demo

## Depends on

* truffle
* ganache-cli / ganache-ui
* react
* ethers
* @mui/material

## Getting started

```
    $ git clone https://github.com/ChanMo/voting
    $ cd voting
    $ npx ganache-cli # or use ganache-ui
    $ npx truffle build
    $ npx truffle migrate
    $ cd client
    $ npm install
    $ npm run start
```
